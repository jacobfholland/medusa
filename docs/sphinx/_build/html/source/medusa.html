medusa.app.run() → None

Run the Medusa Application.

This function starts the application based on the configuration in the object. It enables or disables the database and server components based on the environment variables.

Raises:

  • ImportError: If the server package is missing.

  • Exception: If an unexpected error occurs while starting the server.

Returns:

None
